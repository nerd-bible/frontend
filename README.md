# frontend

Interactive Bible study web app packaged as a:
- [Website](https://nerd.bible)
- Native app for:
  - iOS
  - Android
  - macOS
  - Windows
  - Linux

## Why a native app?

1. User-writable Git repos
   - Sync with trusted Git servers (including over SSH)
   - Version your documents and share them
   - Text-based, so you own your files and can use them without this app
2. Persistence
   - Browsers are allowed to evict files at any time
   - Truly offline without finicky service workers
3. Local Git or web hosting
   - Share with people on your network
4. Bluetooth file sharing
   - Share with nearby people without a network connection
